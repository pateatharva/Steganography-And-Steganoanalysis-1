## final_ganstego.pth

This is the GAN-based steganography model file (421.89 MB).

**Note:** Due to GitHub's file size limitations (100 MB), this model file is not included in the repository.

### How to use your existing model:

If you already have the model file locally, ensure it's placed in this directory:
```
backend/models/final_ganstego.pth
```

The application will automatically load it when starting the backend server.

- **Purpose:** GAN-based image steganography encoder and decoder
